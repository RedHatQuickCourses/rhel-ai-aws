Lab: Launch RHEL 9 AWS Instance

In this lab exercise, you’ll set up an AWS RHEL 9 instance, to build a customized RHEL AI AMI to streamline future deployments of RHEL AI cloud instance.

  • Navigate to the EC2 dashboard in the AWS Management Console.

  • Launch a new instance:

    1. Click on the "Launch Instance" button.

  • Configure instance settings:

    1. Enter "rhel9" as the instance name.

    2. Select RHEL 9 AMI for this new instance:

    3. Select "t3.large" as the instance type.

    4. Create and download an SSH Key for passwordless access to the instance.

    5. Select the newly created VPC and subnet and enable to the Auto-assign public IP option.

    6. Allow incoming traffic on port 22 (SSH) to enable secure access to your instance.

    7. Enter 100GB for storage size.

    8. Review your instance settings, then click "Launch" to create your instance.

    9. Once the instance is running, use the downloaded SSH key to connect to the instance via an SSH client.

      1. Change the permission of the downloaded ssh key to 600.

        $ chmod 600 ~/Downloads/rhel9.pem

      2. Take ssh of user ec2-user using the public IP address of the instance.

        $ ssh -i ~/Downloads/rhel9.pem ec2-user@3.147.72.177
